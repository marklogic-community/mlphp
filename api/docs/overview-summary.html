MLPHP

MLPHP is a PHP API for MarkLogic that makes it easy to store XML, JSON, binary, and text documents, manage document metadata, and create sophisticated search queries on a web server running PHP (version 5.3 or greater). The PHP classes communicate with the MarkLogic via the [MarkLogic REST API.
